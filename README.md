# 🎮 تريدات روبلوكس

موقع تداول أغراض روبلوكس مع أنيميشن احترافي وتخزين محلي.

## 🚀 النشر على GitHub Pages

1. أنشئ repository جديد على GitHub
2. ارفع جميع الملفات
3. اذهب إلى Settings > Pages
4. اختر الفرع الرئيسي (main) والمجلد الجذر (root)
5. انتظر دقيقتين ثم افتح الرابط

## ✨ المميزات

- ✅ تسجيل دخول وتسجيل حساب
- ✅ نشر تريدات مع صور
- ✅ ربط بحساب TikTok
- ✅ أنيميشن احترافي وواقعي
- ✅ تخزين محلي (LocalStorage)
- ✅ تصميم متجاوب
- ✅ تأثيرات بصرية خرافية

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3 (Animations, Grid, Flexbox)
- JavaScript (ES6+)
- LocalStorage API
- GitHub Pages
